Centos中运行jar包的几种方法

1.直接运行
2.后台运行

以Centos 6.5 为例

先通过

命令 进入jar包项目所在目录

运行通过maven打包好的jar包

该项目运行在服务器的443端口

demo-0.0.1-SNAPSHOT.jar

直接运行

当前ssh窗口被锁定 这个期间无法对服务器执行其他命令

若想终止进程 则按下Ctrl+C
或者关闭ssh窗口

后台运行

后台运行分几种情况

普通后台运行

&代表后台运行

当前ssh窗口不被锁定

关闭ssh窗口则程序终止

nohup方式

nohup方式会不挂断运行命令

当账户退出或终端关闭时,程序仍在后台运行

当用 nohup 命令执行作业时，该作业的所有输出会默认被重定向到nohup.out的文件中，除非另外指定了输出文件。

若想重新制定输出文件

这样输出内容不打印到屏幕上，而是输出到temp.txt文件中。

可通过jobs命令查看后台运行任务

此时会列出所有后台执行的作业

且每个作业前面都有个编号。

[1] xxxx
[2] xxxx

若想将某个作业调回前台控制，只需要 fg + 编号即可。

若关闭了ssh窗口 下次重新连接服务器时

可通过查询项目所在端口查询进程PID

再用kill命令杀死进程
